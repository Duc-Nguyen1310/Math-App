## Run Locally

Clone the project

```bash
  git clone https://github.com/Duc-Nguyen1310/Math-App.git
```

Go to the project directory

```bash
  cd Math-App
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
